# Getting started

In order to install the library, follow the next steps:

```sh
# Clone this repository
git clone git@github.com:DevoInc/genesys-ui-form-builder.git

# Go into the repository
cd genesys-ui-form-builder

# Install dependencies
npm ci

# Run the development environment
npm start
```

# Presets

Presets are prefabricated configurations for a repetitive groups of fields.

```js
import { FormBuilder, getFontWeight } from '@devoinc/genesys-ui-form-builder';

<FormBuilder
  model={{
    children: [
      getFontWeight()
    ]
  }}
/>
```

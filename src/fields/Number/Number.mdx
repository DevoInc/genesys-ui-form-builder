## Validation

We can add `validation` with a function that returns a validation object

```js
{
  result: true, // or false
  status?: 'error', // the allow status are: 'success', 'warning', 'error' or
                   // 'info', or nothing in case no state neeeded
  message?: 'text' // The validation message to show
}
```

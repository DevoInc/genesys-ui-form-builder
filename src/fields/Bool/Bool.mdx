## Description

The main prop is `value`, if you use `true` then the switch will be on
the **on** side, otherwise will be on the **off** side (If no `name` is defined
then the field is align to the left)

<Canvas of={Stories.Value} />

You can use props `name` and `helper` for add info to the component

<Canvas of={Stories.Decorators} />

compatible values like `5rem` or `5px`
